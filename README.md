# Dev-Detective-App
Dev Detective is a web application that allows users to search and explore GitHub developer profiles in a clean and interactive interface. The app fetches real-time data from the GitHub API and displays detailed user information, including repositories, followers, bio, and other key metrics.
🚀 Features

🔍 Search any GitHub username

👤 View developer profile details (name, bio, avatar, location, etc.)

📦 Display public repositories

⭐ Show followers and following count

🗓️ Account creation date

🌙 Light/Dark mode support (if implemented)

⚡ Real-time data fetching using GitHub API

📱 Responsive design for mobile and desktop
